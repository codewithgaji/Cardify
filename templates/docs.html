NIN Info Endpoint

URL: /nin-info/

Methods: GET, POST

Description: Retrieve all NIN ID Card records with GET; create a new NIN record with POST.

Example GET Request

GET /nin-info/ HTTP/1.1
Host: cardify-api.onrender.com

Example POST Request

POST /nin-info/ HTTP/1.1
Host: cardify-api.onrender.com
Content-Type: application/json

{
  "first_name": "John",
  "middle_name": "Doe",
  "last_name": "Smith",
  "nin": "12345678901",
  "dateofbirth": "2000-01-01",
  "address": "123 Main St",
  "state_of_origin": "Lagos",
  "country": "Nigeria",
  "gender": "M"
}

Business Info Endpoint

URL: /business-info/

Methods: GET, POST

Description: Retrieve all Business ID records with GET; create a new Business record with POST.

Example GET Request

GET /business-info/ HTTP/1.1
Host: cardify-api.onrender.com

Example POST Request

POST /business-info/ HTTP/1.1
Host: cardify-api.onrender.com
Content-Type: application/json

{
  "full_name": "Acme Corporation",
  "job_title": "CEO",
  "company_name": "Acme Inc",
  "phone_number": "08012345678",
  "email_address": "info@acme.com",
  "website": "https://www.acme.com",
  "gender": "F"
}

License Info Endpoint

Methods: GET, POST

Description: Retrieve all Driver's License records with GET; create a new Driver's License record with POST.
