NIN ID Card Endpoint

URL: /nin-info/

Methods: GET, POST

Description: Use GET to retrieve all NIN ID Card records; use POST to create a new record.

Example Requests

GET /nin-info/ HTTP/1.1
Host: https://cardify-api-by76.onrender.com/

POST /nin-info/ HTTP/1.1
Host: https://cardify-api-by76.onrender.com/
Content-Type: application/json

{
  "full_name": "John Doe",
  "profile_pic": "image_url",
  "gender": "M",
  "address": "123 Main St",
  "nin": "12345678901",
  "dateofbirth": "2000-01-01",
  "country": "Nigeria",
  "state_of_origin": "Lagos"
}

To interactively test this endpoint using a form, please visit /test-form/.
